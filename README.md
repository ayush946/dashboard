# DashBoard using Flask , React and d3.js

# How to run

## Server

nevigate to server/.venv

In windows powershell :
``` .\Scripts\Activate.ps1 ``` 
In Linux
```./Scripts/activate ```

it will start the virtual Environment

To run the server

``` flask -app App.py run ```


## React app

nevigate to dashboard

to run the app
``` npm start ```
